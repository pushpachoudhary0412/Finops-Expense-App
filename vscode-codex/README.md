# Codex (minimal)

This is a tiny VS Code extension that adds a "Codex" container to the Activity Bar and a simple webview view.

How to run

1. In VS Code, open the folder `vscode-codex` (File → Open Folder... → select this folder).
2. Press F5 to open a new Extension Development Host window.
3. In the new window the Activity Bar should show a "Codex" icon (text). Click it to open the Codex view.

Notes

- This is a minimal scaffold. Replace the `extension.js` webview HTML with your UI.
- Update the `publisher` field in `package.json` before publishing.
